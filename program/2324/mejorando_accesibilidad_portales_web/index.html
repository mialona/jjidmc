Mejorando la accesibilidad en portales web de la administración

Miren Mirari San Martín Lacunza

Accesibilidad web, Subtitulado de imágenes, Transcripción de vídeos, Legibilidad, Simplificación de textos, Procesamiento del Lenguaje Natural.

La accesibilidad de las páginas web de las instituciones públicas es de gran importancia para garantizar la igualdad de acceso a la información y los servicios a todas las personas, incluidas aquellas con discapacidad. En este proyecto pretendemos mejorar la accesibilidad de la página web de las administraciones públicas.

En concreto, nos centramos en tres aspectos diferentes que implican la aplicación de distintas técnicas de procesamiento del lenguaje natural. En primer lugar, subtitularemos automáticamente todas las imágenes de la página web; en segundo lugar, proporcionaremos transcripciones de todos los vídeos del Gobierno de La Rioja; y, por último, mejoraremos la legibilidad de los contenidos de la página web. En resumen, este proyecto es un primer paso para que las páginas web de las instituciones públicas se adapten a las necesidades de cada usuario que las visita.
